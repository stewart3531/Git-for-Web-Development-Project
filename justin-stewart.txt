    1. What is Semantic HTML?
    HTML tags with self descriptive vocabulary.

    2. What is HTML used for?
    HTML is used to build the foundations of every website. It is used to display and section all of the core information.

    3. What is an attribute and where do we put it?
    HTML attributes give tags additional information that reference a specific value.

    4. What is the h1 tag used for? How many times should I use it on a page?
    An h1 tag is for a header, h1 should only be used once as it represents the most important heading.

    5. Name two tags that have required attributes
    The <a> and <img> tag both require attributes to specify a location to reference.

    6. What do we put in the head of our HTML document? 
    <!DOCTYPE html>

    7. What is an id?
    An id is an attribute associated with a tag that refrences a specific name.

    8. What elements can I add an id to? 
    Any element can have an id, but it must be unique and used only once.

    9. How many times can I use the same id on a page?
    1 time only. 

    10. What is a class? 
    Classes can be assigned to html tags to reference for styling purposes in CSS.

    11. What elements can I add a class to?
    Any element can have a class.

    12. How many times can I use the same class on a page?
    Unlimited.

    13. How do I get my link to open in a new tab?
    You will need to add the target="_blank" attribute to your anchor tab following your url.

    14. What is the alt attribute in the image tag used for?
    In case the image canot render on a specific users browser, or in the case of using a screen reader, the alt message will be displayed or read.

    15. How do I reference an id?
    <tag id="id-name"></tag>

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    A class is used to select multiple elements and give them the same styling attributes. For example all lists might have a class of <ul class="list-info">. An id would be used to give specificity to only one element that is unique, for example <h1 id="main-heading">.
